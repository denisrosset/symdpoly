    Base class for a generalized free monoid.

    Base class for a generalized free monoid.

    This class must be used as follows.

    - Declare inner case classes extending Op for each type of operator involved in the ring. - Call declare for each operator with the relevant range of indices. - Implement the abstract method "adjoint".
