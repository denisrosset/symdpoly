Companion object FreeBasedMonoidDef

abstract class FreeBasedMonoidDef extends MonoidDef

Monoid whose elements are represented by normal forms in a free monoid. Is not necessarily a strict quotient monoid, as free.MonoidDef inherits from this; then code can be reused between free monoids and their quotients.
