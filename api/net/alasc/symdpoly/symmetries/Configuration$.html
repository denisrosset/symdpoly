  4.  def apply(n: Int, generators: Seq[GenPerm]): Configuration

    Use an orbit enumeration algorithm to generate a configuration from a group of generalized permutations.

    Use an orbit enumeration algorithm to generate a configuration from a group of generalized permutations.

    We use the following convention. Let X(r,c) be a cell of the configuration matrix, and g a generalized permutation. We must have X(r1, c1) = X(r,c) * pr.reciprocal * pc, where (pr, r1) is the image of (Phase.one, r) under g, and (pc, c1) is the image of (Phase.one, c) under g.
