  4.  trait Phased[A] extends MultiplicativeAction[A, Phase]

    Describes an element with a phase.

    Describes an element with a phase. Each element a \in A is associated to an equivalence class (up to phase) written [a] = {a <* phase | phase \in AllPhases }.

    Then phaseCanonical(a) returns a canonical representative of the equivalence class [a], and phaseOffset(a) returns the phase of a relative to phaseCanonical(a) such that

    a === phaseCanonical(a) <* phaseOffset(a)
