  14.  def momentIndexMatrixDescription: String

    Writes the Gram matrix indices to the file with given filename.

    Writes the Gram matrix indices to the file with given filename. The structure of the file is as follows

    Line 1: N M Line 2: g11 g12 ... g1N ... Line 1+N: gN1 gN2 ... gNN

    where N is the size of the Gram matrix, and M is the number of monomials involved, including the identity.

    The matrix index g(r,c) takes the value

    - 0 if the element is always 0 - 1 if the element takes the constant value 1 or -1 - j if the element corresponds to the expectation value of j-th monomial (with a possible factor -1)

    and max_{r c} g(r,c) = M
