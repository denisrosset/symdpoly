Companion object Block

case class Block(size: Int, basisSize: Int, basisIndices: Array[Int], rowIndices: Array[Int], colIndices: Array[Int], coefficients: Array[Double]) extends Product with Serializable

Describes a SDP constraint block expressed as a linear combination of basis elements on real scalar variables.

The block is described as a sparse tensor by a sequence of (basisIndex, rowIndex, colIndex, coefficient) tuples.

size

The block is a matrix of shape size x size

basisSize

Number of elements

basisIndices

Index of basis element

rowIndices

Row index

colIndices

Column index

coefficients

Real coefficient
