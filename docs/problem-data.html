Exporting structured data

Again, we work with the CHSH relaxation.

scala> import net.alasc.symdpoly.examples.quantum.CHSH
import net.alasc.symdpoly.examples.quantum.CHSH

scala> val relaxation = CHSH.relaxation
relaxation: net.alasc.symdpoly.Relaxation[net.alasc.symdpoly.evaluation.Evaluator.<refinement>.type,net.alasc.symdpoly.examples.quantum.CHSH.Lsym.Mono] = Moment relaxation with 1 monomials, a moment matrix of size 5 x 5, and 0 localizing matrix/ces

Exploring the monomial matrix structure

To explore the moment matrix, inspect:

scala> relaxation.momentMatrix
res0: net.alasc.symdpoly.MomentMatrix[net.alasc.symdpoly.evaluation.Evaluator.<refinement>.type,net.alasc.symdpoly.examples.quantum.CHSH.Lsym.Mono] = net.alasc.symdpoly.MomentMatrix@764d2bed

Note that the monomials are included into brackets, as they represent equivalence classes of monomials in the quotient algebra.

This description works only for smaller moment matrices.

The method allMoments returns a list of the monomials present.

scala> relaxation.allMoments
res1: net.alasc.symdpoly.util.OrderedSet[net.alasc.symdpoly.examples.quantum.CHSH.Lsym.EvaluatedMonomial] = OrderSet(L([1]), L([A(0) B(0)]))
